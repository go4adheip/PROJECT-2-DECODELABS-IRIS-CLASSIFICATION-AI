# Iris Flower Classification Using Machine Learning

## Project Overview
This project uses the Iris Flower Dataset to classify flower species using the K-Nearest Neighbors (KNN) Machine Learning algorithm.

The model is trained on flower measurements and predicts whether a flower belongs to:
- Setosa
- Versicolor
- Virginica

## Technologies Used
- Python
- Pandas
- Scikit-Learn
- Matplotlib
- Seaborn

## Dataset
The Iris dataset contains:
- 150 flower samples
- 4 features:
  - Sepal Length
  - Sepal Width
  - Petal Length
  - Petal Width
- 3 flower species

## Machine Learning Workflow
1. Load Iris Dataset
2. Explore Dataset
3. Split Data into Training and Testing Sets
4. Scale Features
5. Train KNN Classifier
6. Make Predictions
7. Evaluate Model Performance
8. Visualize Confusion Matrix

## Results
- Accuracy: Approximately 95% - 100%
- Classification Report Generated
- Confusion Matrix Visualized
